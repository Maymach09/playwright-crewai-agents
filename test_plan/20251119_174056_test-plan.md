# Application - Feature Test Plan
## Application Overview 
This test plan outlines the steps to edit an existing account in Salesforce, specifically updating the account name and phone number.

## Test Scenarios
### 1. Edit Existing Account
**Seed:** tests/seed.spec.ts
#### 1.1 Update Account Name and Phone Number
**Steps:**
1. Navigate to the Salesforce Accounts module.
2. Click on the "Accounts" link in the navigation.
3. Select the account "Test Account 2" from the list.
4. Click the "Edit" button to modify account details.
5. Type "Updated Test Account 2" into the "Account Name" textbox.
6. Type "123-456-7890" into the "Phone" textbox.
7. Click the "Save" button to update the account details.

**Expected Results:**
- Step 1: The Accounts module should load successfully.
- Step 2: The list of accounts should be displayed.
- Step 3: The account details for "Test Account 2" should be shown.
- Step 4: The edit form should be displayed.
- Step 5: The account name should be updated to "Updated Test Account 2".
- Step 6: The phone number should be updated to "123-456-7890".
- Step 7: A success notification should appear confirming the account has been updated, and the account details should reflect the changes made.
